# Ambient Granulator for 3DS v0.1

This is the first public native Nintendo 3DS/2DS release of Ambient Granulator.
It preserves the immediate tracker-style controls and waveform interaction of
the Nintendo DS instrument while moving synthesis, effects, input, and storage
onto the native 3DS homebrew stack.

## Highlights

- Native Homebrew Launcher `.3dsx`, tested on Nintendo 2DS hardware and Azahar.
- Ten embedded rights-cleared samples, converted from the supplied originals to
  48 kHz mono PCM16 and spatialised per grain at runtime.
- Polyphonic granular engine with touch/A gating, B bursts, pitch and fine
  deviation, envelopes, timing jitter, and free or BPM-synchronised triggering.
- Stereo FDN reverb with long delay times, 99.9% maximum feedback, and true
  Freeze; switching samples no longer interrupts the effects tail.
- Slow pre-reverb stereo phaser, stereo HPF/LPF, final soft clipping, and output
  metering.
- Adjustable native microphone gain and an always-live input meter, with
  destructive R-held punch-in beginning at the current waveform position.
- D-pad, Circle Pad, and New 3DS C-stick navigation/editing with held repeat;
  hold Y plus Left/Right to move the waveform position.
- Standalone browser patcher for replacing the sample bank directly inside the
  `.3dsx` without a server or separate runtime data file.

## Download

Most users should download `ambient-granulator-for-3DS-v0.1-sd.zip`, extract
it, and merge its `3ds` folder into the root of the console SD card. The direct
`.3dsx` binary is also provided for manual installs and browser patching.

The release also includes `SHA256SUMS.txt` for artifact verification.

## Requirements

- A Nintendo 3DS-family system able to run Homebrew Launcher applications.
- DSP firmware configured through the maintained 3DS Hacks Guide setup process.
- Headphones are recommended when recording to avoid punching speaker output
  back into the console microphone.

## Known limitations

- OSC networking and camera-driven control are planned but not included.
- No CIA package is provided.
- Emulator microphone routing and latency are not authoritative; physical
  hardware remains the reference for microphone and timing behaviour.

See the repository Quick Start for installation and control details.
